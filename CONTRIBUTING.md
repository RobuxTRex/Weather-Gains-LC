## Code of Conduct
Before making any contributions, please read our [Code of Conduct](./CODE_OF_CONDUCT) to ensure a welcoming environment for everyone! Failure to follow the Code of Conduct will result in disciplinary action.

---

## Contributor License Agreement (CLA)
You must agree to the [Contributor License Agreement (CLA)](./CLA.md) before submitting code to this repository. If you do not agree to these terms, do not submit contributions.

---

## Reporting Issues
Issues can and will occur in WeatherGains, especially due to the complete code refactor found in v2.x.x! If needs be, you can report issues or suggest new features through the [GitHub Issues](https://github.com/RobuxTRex/Weather-Gains-LC/issues) tab. 

Please ensure you include clear steps to reproduce bugs, and explain the expected vs. actual behaviour. Attaching screenshots or code snippets where possible helps greatly!

---

## Branching Model
- Work on feature branches (`feature/<name>`) and submit pull requests to `main`.
- The `legacy` branch is archival (v1.x.x) and must not receive new commits.
